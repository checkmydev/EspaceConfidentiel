# LUMENA — Espace Confidentiel

Site de démonstration en HTML/CSS/JS vanilla, inspiré de l'esthétique de landing page
immobilière de luxe de [thecapston.com](https://thecapston.com) (fond sombre, typographie
serif dorée, mise en page épurée).

**Tout le contenu (nom de résidence, adresse, prix, prestations) est fictif** et généré
à titre d'exemple. Aucune donnée réelle n'est utilisée.

## Structure

- `index.html` — page unique (hero, présentation, prestations, résidences, galerie,
  emplacement, formulaire d'inscription)
- `css/style.css` — thème sombre/or, typographies Cormorant Garamond + Jost
- `js/script.js` — menu mobile, effet de scroll sur le header, animations d'apparition,
  formulaire de démonstration (sans backend)

## Lancer en local

Ouvrir simplement `index.html` dans un navigateur, ou servir le dossier avec un serveur
statique quelconque (ex. `python3 -m http.server`).
